# 🏛️ Museu Nacional

## 📌 Sobre o projeto

Este é um site institucional desenvolvido com o objetivo de apresentar informações sobre o Museu Nacional, incluindo história, exposições e conteúdos visuais.

## 🎯 Objetivo

Este projeto foi criado como parte de um curso de desenvolvimento web, com foco na prática de HTML e CSS, aplicando conceitos de estruturação de páginas e estilização.

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3

## 🌐 Acesse o projeto

👉 https://julioclsantos.github.io/museu-nacional/
